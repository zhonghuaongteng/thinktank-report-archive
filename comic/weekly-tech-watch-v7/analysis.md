# 分析口径

## 用户反馈对应问题

前一版画面有冲突感，但议题锚点仍不够明确。读者能看到“堵点”和“上海参考”，却不容易判断这对应哪份报告、报告在说什么、政策建议落在哪里。

## v7 主判断

本版改为“单篇报告证据链”结构。画面只服务于一条判断：出口管制正在从分散措施转为可被主动校准的经济安全工具，并通过关键矿产、材料加工设备和下游产业链传导风险。

## 画面任务

1. 显示政策工具成形：出口管制从临时反制走向制度化工具箱。
2. 显示传导机制：关键矿产和材料加工设备成为供应链闸门。
3. 显示政策建议：持续监测贸易和投资数据、开展供应链压力测试、准备替代来源。
4. 显示上海参考：半导体、新能源汽车、先进制造和清洁技术需要把材料风险、出口合规和产业链韧性纳入日常监测。

## 视觉原则

- 采用一页四格横向叙事，避免泛主题封面。
- 画面主体是“政策工具如何影响供应链”，不使用抽象科技背景。
- 中文标签控制在 8 个以内，保证可读性。
- 精确观点放入周报“读图说明”，图像承担冲突现场和证据链提示。
